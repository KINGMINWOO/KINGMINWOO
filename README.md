#### 프로젝트 개요

- 각종 센서(흔들림 감지 센서, 적외선 감지 센서 등)와 모듈(RFID, 서보모터 등)을 TCP/IP를 통해 상호 연결하여 스마트 오피스 & 재난 대비 시스템 구현
